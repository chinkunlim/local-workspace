4. **Feynman Technique**: Force the model to invent a *novel, everyday physical analogy* (not mentioned in the text) to explain the hardest abstract concept in the document.

## 2. Walkthrough / Summary

# Verification Dashboard Walkthrough

Based on your feedback, we've entirely redesigned the Proofreader's Verification Gate! The system is now a non-blocking, centralized **Dashboard** that lets you review AI corrections on your own schedule.

## 🌟 Key Features

### 1. Non-Blocking Automation (非同步背景執行)
The `proofreader` pipeline no longer pauses the entire system. Instead, it saves the Ollama-corrected Markdown directly into a "Pending Review" queue and moves on to the next file. You can now process an entire subject without interruption.

### 2. Side-by-Side Review UI (Ground Truth 預覽)
The UI has been simplified and optimized for human reviewers.
- **Left Pane (Original File)**: Directly embeds the original PDF document, PNG image, or M4A audio player. You see the *actual* source material, not just raw extracted text.
- **Right Pane (Markdown Editor)**: A powerful Monaco text editor containing the AI's proofread results.

### 3. Subject Grouping (依學科分類)
The left sidebar automatically groups all pending and completed reviews by Subject (e.g., `助人歷程`). It displays badges to indicate whether it's a Document Proofread (`Doc`), Transcript (`Tx`), or Completeness check (`Comp`).

### 4. Re-Verification (隨時重複校對)
Any verified file can be re-opened and re-edited at any time. Simply click the file in the sidebar, modify the Markdown, and hit `Cmd+S` or click `Save & Mark Completed`. The file in `data/proofreader/output/` is instantly updated.

## 🚀 How to Use It

1. First, ensure the `dashboard.py` script is running. (We are currently running it on `http://localhost:8080`).
2. Open your web browser and navigate to **http://localhost:8080**.
3. Select any file from the sidebar under the `助人歷程` group.
4. Compare the original file on the left with the AI text on the right. Make any edits you wish!
5. Click **Save & Mark Completed** to finalize your review.
